# Snake (React)

Simple Snake game built with React + Vite.

Quick start

1. Open a terminal in `snake-react` folder
2. Install dependencies:

```
npm install
```

3. Run dev server:

```
npm run dev
```

Controls: arrow keys to move, `Space` to pause/resume.
